What Are Wireframes?

Wireframes are a display or layouts of a website page. We use them to plan to create our applicaton. It could be use as a prototype or demo to what the final product may be like. It is also the structure or skeleton to our webpage.

Git Workflow

Git Workflow works when a repository is created and shared between members. Members can push their work to the master branch, If accepted by the project manager. He/she will accept the pull request of the member's work and merge it with the master branch which is also the final product. When a member is behind and need the updated work from the master branch. The member can pull from master branch and merge it with his/her work.
